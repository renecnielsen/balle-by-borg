-balle, -by & -borg

A visual exploration of the spatial patterns in the endings of Danish town and village names. It's a 99.8% copy of the amazing Moritz Stefaner's amazing German original (his code/my code). If you find this just slightly interesting, please go have a look at Moritz' original. The German version is actually much more interesting.

I used suffixes from https://da.wikipedia.org/wiki/Danske_stednavne with ~50 or more uses, and used Moritz' code to cross-reference them with a list of place names from geonames. Please note that darker colors indicate a higher proportion of place names with the respective suffix in a given geographic cell. Hover over a cell for a while to see a tooltip with the place names.

René Clausen Nielsen (@ReneCNielsen), 2016
